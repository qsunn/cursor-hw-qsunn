• git config --global user.name blabla - змінити ім'я
  • git config --global user.email blabal@bla - змінити мейл
  • git init - створення локального репозиторія
  • git status - перевірка статусу
  • git add - збереження змін
  • git commit - повідомлення про зміни
  • git log - показ усіх коммітів від новіших до старіших
  • git checkout - переключення гілок
  • git branch - управління гілками
  • git push - закинути дані на віддалений репозиторій
  • git pull - забрати дані з віддаленого репозиторія
  • git merge - об'єднання гілок
  • git clone - клонувати проект з віддаленого репозиторія

.gitignore - файл, який визначає, які файли та папки слід ігнорувати у заданому вихідному коді.

  • git revert - відкат комміту створенням нового
  • git reset - відміна локальних змін в репозиторії
  • git rebase - це як git merge
  • git cherry-pick - перенесення конкретних коммітів з одного місця репозиторія в інше
  • git stash - ховає зміни у файлах та зберігає ці зміни окремо, щоб потім можна було їх повернути
